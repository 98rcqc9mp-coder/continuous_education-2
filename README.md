# نظام التعليم المستمر — continuous_education-2

## نظرة عامة
نظام PHP + MySQL لإدارة الدورات التدريبية والمشاركين وإصدار الشهادات.

---

## المتطلبات
- PHP 7.4 أو أعلى
- MySQL 5.7 / MariaDB 10.4 أو أعلى
- خادم ويب (Apache / Nginx)

---

## طريقة تطبيق الترقية (2026-03-09)

### 1. تطبيق ملف قاعدة البيانات
```bash
mysql -u root -p continuous_education < sql/upgrade_2026-03-09.sql
```
أو قم باستيراد الملف عبر phpMyAdmin.

يضيف الملف:
- عمود `participants.email`
- عمود `courses.min_attendance_pct`
- جدول `certificates` — سجلات الشهادات
- جدول `course_registration_requests` — طلبات التسجيل
- جدول `rooms` — القاعات التدريبية
- جدول `course_room_schedule` — جدولة القاعات
- جدول `attendance` — الحضور اليومي

### 2. إعداد البريد الإلكتروني (اختياري)
افتح `config/mail.php` وبدّل `enabled => true` وأدخل بيانات البريد.

---

## الميزات الجديدة

### 1. نظام التحقق من الشهادات
- عند فتح شهادة مشارك (`certificate.php`) يُنشأ تلقائياً رمز تحقق فريد + QR.
- صفحة عامة للتحقق: `verify_certificate.php?code=XXXX`
- يعرض: اسم المشترك، اسم الدورة، التواريخ، اسم الجهة المانحة.

### 2. النسخ الاحتياطي والاسترجاع
- الصفحة: `admin/backup.php`
- زر تنزيل ملف `.sql` كامل.
- رفع ملف `.sql` لاسترجاع البيانات.
- محمية بـ CSRF ويتطلب دور admin.

### 3. إرسال بريد عند التسجيل
- عند إضافة مشارك من `add.php` مع تعبئة حقل البريد الإلكتروني.
- الإرسال معطّل افتراضياً (فعّله من `config/mail.php`).

### 4. تسجيل مشترك عبر الإنترنت + موافقة الإدارة
- صفحة عامة: `public/register_course.php`
- صفحة الأدمن: `admin/requests.php`
- عند الموافقة: يُضاف المشارك تلقائياً ويُرسل بريد تأكيد.

### 5. إدارة القاعات ومنع التعارض
- الصفحة: `admin/rooms.php`
- إضافة/حذف القاعات (الاسم، السعة، الموقع).
- جدولة دورة في قاعة مع التحقق من عدم التعارض الزمني.

### 6. الحضور اليومي + شرط الشهادة
- الصفحة: `admin/attendance.php?course_id=X`
- تسجيل حضور/غياب لكل مشارك لكل يوم.
- تحذير في الشهادة إذا نسبة الحضور < الحد الأدنى (افتراضي: 80%).

```

  mail.php        ← إعدادات البريد الإلكتروني
lib/
  auth.php        ← دوال التحقق من الجلسة
  csrf.php        ← دوال حماية CSRF
  qr.php          ← توليد رمز QR
  mail.php        ← دوال إرسال البريد
sql/
  upgrade_2026-03-09.sql  ← ملف ترقية قاعدة البيانات
admin/
  backup.php      ← نسخ احتياطي / استرجاع
  requests.php    ← إدارة طلبات التسجيل
  rooms.php       ← إدارة القاعات وجدولتها
  attendance.php  ← سجل الحضور اليومي
public/
  register_course.php  ← نموذج التسجيل العام
verify_certificate.php  ← صفحة التحقق من الشهادة (عامة)
```
